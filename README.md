## Overview
BlockFund is a decentralized application built on Ethereum that enables ERC-20 based crowdfunding, optional sponsorship, and fair token distribution to beneficiaries.

## Features
- ERC-20 token with fixed price
- Multi-campaign crowdfunding support
- Sponsor-based bonus funding
- Weighted fund distribution
- Role-based access (owner, contributor, shareholder)

## Tech Stack
- Solidity
- React + ethers.js
- Hardhat / Remix
